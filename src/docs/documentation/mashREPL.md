# mashREPL
!!! info "What's mashREPL? And why this name?"
    It's a small **R**ead **E**val **P**rint **L**oop that allow you to run *local* commands inside your iOS and iPadOS device!

    **MASH** stands for : **M**engus **A**rnaud **SH**ell
