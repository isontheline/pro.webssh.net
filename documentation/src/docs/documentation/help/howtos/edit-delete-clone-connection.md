---
title: Edit / Delete / Clone a connection (server)
---

This guide will help you to edit a server when you need to change its settings, delete a server you no longer need, or clone a server to create a duplicate with the same settings.

# How to edit, delete or clone a connection (server)?
## on iOS
1. Long press the connection item (server) line
2. Choose the action you want : Edit, Delete or Clone

## on macOS
1. Right click on the connection item (server) line
2. Choose the action you want : Edit, Delete or Clone
