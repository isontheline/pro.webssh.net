# 32 - Comet :comet:
!!! quote "Comet"
    A comet is an icy, small Solar System body or interstellar object that warms and begins to release gases when passing close to the Sun, a process called outgassing. This produces an extended, gravitationally unbound atmosphere or coma surrounding the nucleus, and sometimes a tail of gas and dust gas blown out from the coma.

    [Wikipedia](https://en.wikipedia.org/wiki/Comet)
